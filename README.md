# 真実の扉（Proposition Dungeon）

ブラウザだけで遊べる、命題の真偽を扉で選ぶ進行型ミニゲームのプロトタイプです。疑似 3D の通路と扉演出、正解で宝石ゲット・不正解で奈落に落ちるエフェクトを入れています。

## 遊び方

1. `index.html` をブラウザで開きます（ローカルでも OK、必要に応じて `python -m http.server 3000` などで簡易サーバを立ててください）。
2. 扉の前で表示される命題に対し、「真」「偽」の扉をクリックします。
3. 正解なら宝石と解説が表示され、次の扉へ進むか逃げ出すかを選べます。誤答すると奈落に落ちてゲームオーバー（解説付き）。
4. 連続で何個の宝石を集められるか挑戦してください。

## カスタマイズ

- `script.js` 内の `questions` 配列を書き換えると問題・解説・正誤を差し替えられます。
- ビジュアルや色は `style.css` の CSS 変数を変えるとまとめて調整できます。

## 構成

- `index.html` … レイアウトとUI
- `style.css` … シーン演出・アニメーション・タイポグラフィ
- `script.js` … フロー管理（扉選択→判定→結果表示）とスコア更新
